# BMAD Shared Tools

This package contains reusable BMAD automation tools for OpenSwarm.

## Included tools

- `BmadInstantiateTemplate`
- `BmadValidateArtifact`
- `BmadValidateDirectory`

## Purpose

These tools operationalize BMAD template generation and artifact validation so BMAD agents can create and check structured software-delivery artifacts instead of relying on prompts alone.
